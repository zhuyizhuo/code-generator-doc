# 简介

code-generator 是一个 MyBatis 的 ORM 生成工具，一键生成 mybatis xml + model + mapper，为简化开发、提高效率而生。

# 特性

  • 简单轻量
  • 配置化
  • 可扩展
  • 可定制自己的模板

# 支持数据库

  • mysql
  • oracle

# 所需环境

  • JDK 1.8 +
  • Maven or Gradle
